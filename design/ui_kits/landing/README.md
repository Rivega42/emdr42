# UI kit — Главная страница EMDR42

Рекреация главной страницы emdr42.ru в новом арт-дирекшене «Лунная ночь» (Куинджи).
Контент и структура — из `app/page.tsx` репозитория [Rivega42/emdr42](https://github.com/Rivega42/emdr42);
визуальный слой — новая система (см. корневой `readme.md`).

## Файлы
- `LandingPage.jsx` — секции: `LandingNav`, `LandingHero` (сцена + лунная дорожка + дисклеймер),
  `LandingFeatures` (3×2), `LandingCta` (дорожка завершается у кнопки), `LandingFooter`.
  Композирует компоненты системы (`Button`, `Card`, `Disclaimer`) — не переопределяет их.
- `landing.css` — лейаут-стили страницы (классы `l-*`). Не входит в `styles.css` — подключайте отдельно.
- `index.html` — тёмная тема («ночь»), переключатель темы внизу справа. Responsive 1440 → 375.
- `light.html` — светлая тема («рассвет») без переключателя.
- `mobile.html` — превью 375px (iframe вокруг index.html).

## Правила сцены
- Лунная дорожка — один мотив на страницу: от луны в hero вниз, финальный отрезок упирается в кнопку CTA-блока.
- Светится только primary CTA (и регистрация в navbar). Никаких других глоу.
- Мерцание дорожки отключается `prefers-reduced-motion`.
- Иконки — lucide, штрих 1.5px (`lucide.createIcons({ attrs: { 'stroke-width': 1.5 } })`).
