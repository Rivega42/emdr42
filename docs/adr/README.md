# Архитектурные ADR

Полноразмерные ADR в формате `NNNN-title.md`. Шаблон ниже.

## Шаблон

```markdown
# ADR-NNNN: Название

**Статус:** proposed / accepted / deprecated / superseded by ADR-MMMM
**Дата:** YYYY-MM-DD
**Авторы:** @user1, @user2
**Контекст:** что вынудило решать; ограничения, требования
**Опции:** A, B, C — описание каждой
**Решение:** B — потому что …
**Последствия:**
- Положительные: …
- Негативные: …
- Митигация: …
**Связанные:** ADR-NNNN, issue #N, PR #M
```

## Реестр

| ADR | Название | Статус |
|---|---|---|
| _пусто_ | _добавляйте по мере необходимости_ | |
